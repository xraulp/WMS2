## Installation ‚Äî Step by Step

### 1. Create and activate a virtual environment

**Windows (PowerShell):**
```powershell
python -m venv venv
venv\Scripts\Activate.ps1
```

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run database migrations
```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. Create the admin user
```bash
python setup_admin.py
```
Default credentials: **username:** `admin` / **password:** `admin123`
> ‚öÝÔ∏è Change this password after first login via `/admin/`

### 5. Configure your SMTP email settings

Open `warehouse_system/settings.py` and edit the EMAIL section at the bottom:

```python
EMAIL_HOST = 'smtp.tuservidor.com'     # Your SMTP server
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'tucorreo@dominio.com'
EMAIL_HOST_PASSWORD = 'tu_password'
DEFAULT_FROM_EMAIL = 'tucorreo@dominio.com'
```

**Common SMTP providers:**
| Provider | Host | Port |
|----------|------|------|
| Gmail | smtp.gmail.com | 587 |
| Outlook/Hotmail | smtp-mail.outlook.com | 587 |
| Yahoo | smtp.mail.yahoo.com | 587 |
| Office 365 | smtp.office365.com | 587 |

> For Gmail: use an **App Password**, not your regular password.
> Go to: Google Account ‚Üí Security ‚Üí 2-Step Verification ‚Üí App passwords

### 6. Start the development server
```bash
python manage.py runserver
```

Open your browser at: **http://127.0.0.1:8000**

## How the Custom ID Works

| Operation | Date | Result |
|-----------|------|--------|
| 1st ENTRY on Feb 13, 2026 | 260213 | `ED260213-0001` |
| 2nd ENTRY on Feb 13, 2026 | 260213 | `ED260213-0002` |
| 1st EXIT on Feb 10, 2026  | 260210 | `SD260210-0001` |

- **ED** = Entry (Entrada)
- **SD** = Salida (Exit)
- Consecutive resets daily, independently per type.
